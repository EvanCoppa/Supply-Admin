# Runbooks

Operational procedures for pre-launch incidents and routine maintenance.

- [Order cancellation](./order-cancellation.md) — voiding orders before/during fulfillment.
- [Inventory hold release](./inventory-hold-release.md) — clearing stuck `quantity_reserved`.
- [Credential rotation](./credential-rotation.md) — rotating Supabase keys, API tokens, gateway creds.

Each runbook lists when to use it, the procedure, common failure modes, and escalation paths.
